# Customer Churn Analytics Dashboard 📊

A modern, production-ready React application designed to visualize and analyze predictive customer churn data. This project demonstrates advanced frontend architecture, global state management, and test-driven development.

## ✨ Features

* **Complex Data Grid:** A custom-built table featuring sorting, filtering, and pagination without relying on heavy external UI libraries.
* **Global State Management:** Seamless cross-component filtering (e.g., by subscription plan) powered by **Zustand**.
* **Simulated API & Data Caching:** Asynchronous data fetching, loading states, and error handling managed by **TanStack Query (React Query)**.
* **Dynamic Routing:** Individual detailed customer views using **React Router v6**.
* **Data Visualization:** Clean and responsive charts built with **Recharts**.
* **Unit Testing:** Business logic and component rendering tested using **Vitest** and **React Testing Library**.

## 🛠️ Tech Stack

* **Core:** React 18, TypeScript, Vite
* **State Management:** Zustand, TanStack Query
* **Routing:** React Router v6
* **Styling:** Tailwind CSS, Lucide React (Icons)
* **Testing:** Vitest, React Testing Library, jsdom
* **Charts:** Recharts
